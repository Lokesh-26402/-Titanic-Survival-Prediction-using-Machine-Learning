# 🚢 Titanic Survival Prediction using Machine Learning
This project focuses on predicting whether a passenger survived the Titanic disaster using Machine Learning classification algorithms. The project includes data preprocessing, feature engineering, exploratory data analysis (EDA), model training, evaluation, and prediction. Special emphasis is given to handling missing values, extracting meaningful features, and model explainability.
# 🎯 Objectives
- Perform Exploratory Data Analysis (EDA)
- Handle missing values and categorical variables
- Create new features such as titles and family size
- Train and compare multiple classification models
- Evaluate models using classification metrics
- Explain model predictions using feature importance/SHAP
- Save the trained model for future predictions
# Libraries Used
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- SHAP
- Joblib
# Algorithms Used
- Logistic Regression
- Random Forest Classifier
- Decision Tree Classifier
# Features
- Passenger Class (Pclass)
- Gender (Sex)
- Age
- Fare
- Family Size
- Cabin Presence
- Extracted Titles from Names
# Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1-score
- Confusion Matrix
